{% comment %} Item figure This include adds a figure to the page styled using bootstrap, https://getbootstrap.com/docs/4.5/content/figures/ It requires a link to the image file used in the "src" attribute E.G. --> {% include feature/figure.html img="https://www.lib.uidaho.edu/media/spec/collections/oldadmin.jpg" alt="old admin building" %} Options: - "img" = link to image file. - "link" = link to another web page. - "alt" = alt text for image - "width" = will use responsive sizing to set the % size on desktop (will be 100% on mobile), choose from "25", "50", "75", or "100" - "caption" = the caption option allows you to manually add a caption - "float" = will use responsive float utility to add float on desktop (will not float on mobile), choose from "left" or "right" {%- endcomment -%}

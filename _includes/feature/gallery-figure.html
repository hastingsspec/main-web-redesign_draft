{% comment %} Figure include adds an image marked to open up in a gallery viewer. Please ensure to add "gallery: true" to front matter of pages using this include. All items on the page using this include will be browseable in the gallery. Options: - "img" = full url to image - "img-lg" = full url to larger size of the image (optional) - "alt" = description of the image contents for the alt text - "caption" = text for caption of the figure (optional) - "width" = manually set width using options such as "50%" (optional) note: uses Spotlight, https://github.com/nextapps-de/spotlight {%- endcomment -%}
